Ce este Linux?

Pe scurt „Linux” este un sistem de operare alternativ/înlocuitor pentru Windows și/sau macOS care în majoritatea cazurilor este cunoscut ca un sistem foarte greu de folosit, dedicat server-lor și administratorilor.

.Povestea lungă este un pic mai complicată decât pare dar ușor de înțeles dacă ești un pic atent 🙂

În teorie „Linux” este un sistem de operare, dar în practică este nucleul sistemului de operare sau kernel cum este numit în engleză. Numele acestui nucleu/kernel a devenit popular ca un sistem de operare pentru că este ușor de pronunțat, ușor de ținut minte și a avut o reclamă mult mai mare decât numele original, care este GNU/Linux.

Nucleul Linux a fost creat de Linus Torvalds în 1991, care între timp a fost îmbunătățit cu ajutorul programatorilor voluntari din întreaga lume. Acest nucleu a fost lansat sub licența GNU GPL, care este o licență liberă și codul sursă este gratuit și liber, oricine se pricepe îl poate folosi și modifica atât timp cât respectă licența GPL.

Nucleul a fost dezvoltat la început pentru microprocesorul Intel 386 dar acum suportă o mare gamă de microprocesoare și arhitecturi. Este folosit atât pe calculatoare și supercomputere cât și pe sisteme încapsulate ca telefoanele mobile sau bancomate.
