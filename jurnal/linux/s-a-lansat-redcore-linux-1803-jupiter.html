Redcore Linux - Ghidul utilizatorului

S-a lansat Redcore Linux 1803 Jupiter

Pe data de 01.04.2018 chiar de ziua internațională a păcălelilor s-a lansat noua versiune a celei mai populare distribuție GNU/Linux din România, și anume versiune 1803.

Echipa Redcore Linux Project a adus schimbări majore aceste versiune, dar utilizatorul nu v-a simți nimic. Se va simți schimbările doar atunci când se face actualizarea de la versiunea Redcore Linux 1801 la Redcore Linux 1803. Cel mai bine este să se instaleze noua versiune pe curat pentru că durează mult mai puțin decât să se facă actualizarea de la 1801.

Vă întrebați ce schimbări majore aduce Redcore Linux 1803 Jupiter, dacă nu ați citit anunțul oficial vă invit să citiți mai jos caracteristicile cu care vine.

Caracteristicii

  • sincronizat cu Gentoo pe data de 31.03.2018.
  • a fost repartizat pe profilul Gentoo 17.0 profil securizat (hardened).
  • vine cu nucleul LTS 4.14.30 profil securizat.
  • pachete actualizate: GCC 7.3.0, glibc 2.26, binutils 2.30, mesa 18.0.0, clang/llvm 6.0.0, kde-framework 5.44, kde-apps 17.1.3.
  • îmbunătățirea modului de utilizare a procesorului cu până la 300% mai mică în modul „live”.
  • imaginea .iso a fost redusă cu 350 MB.
  • managerul sisyphus (CLI) vine cu două noi opțiuni:
    • mirror list – această comanda afișează lista cu oglinzile depozitului (cea activă este marcată cu * ).
    • mirror set – cu această comandă se schimbă oglinda activă.
  • managerul sisyphus (GUI) face același lucru ca sisyphus (CLI) dar într-un mod mult mai ușor pentru că se face din interfața grafică.
  • pachetul pentru depozit a fost reconstruit cu tehnici de întărire, folosind noul set de unelte (SSP, PIE, RELRO, ASLR).
  • s-a scos câteva aplicații din depozit (yarock, franz, fsarchiver, messengerfordesktop, megasync).
  • s-a adăugat câteva aplicații noi în depozit (etcher, radeon-profile, spotify)

Actualizare de la Redcore Linux 1801 

$ su
# sisyphus update
# eselect profile set default/linux/amd64/17.0/hardened
# emerge -av1 --nodeps vasile
# vasile --binmode
# sisyphus upgrade
# sisyphus remove-orphans

Actualizare de la Redcore Linux 1803 Alpha 

$ su
# sisyphus update
# emerge -av1 --nodeps vasile
# vasile --binmode
# sisyphus upgrade
# sisyphus remove-orphans
# sisyphus mirror set 1

Actualizarea în modul „mixedmode” și „srcmode” nu va funcționa pentru că nu au suport. Aceste moduri pot conține aplicații suplimentare din depozitul Gentoo, programe care nu se afla în depozitul Redcore.
